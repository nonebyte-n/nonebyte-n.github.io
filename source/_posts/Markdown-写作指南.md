# Markdown 写作指南 📝

Markdown 是一种轻量级标记语言，非常适合用于博客写作。

## 常用语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式
- **加粗** `**加粗**`
- *斜体* `*斜体*`
- ~~删除线~~ `~~删除线~~`

### 代码块
```markdown
```python
def hello():
    print("Hello, Markdown!")
```
```

### 表格
| 语法 | 效果 |
|------|------|
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
